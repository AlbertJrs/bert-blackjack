PASOS PARA EJECUTAR PROYECTO

1. CLONAR REPOSITORIO
2. EJECUTAR ```NPM INSTAL``` PARA RECONSTRUIR LOS MODULOS DE NODE
3. CORRER EL DEVSERVER ```NPM RUN DEV```
4. ABRIR EL PROYECTO EN: ```HTTP://LOCALHOST:3000```
